Cross platform Vendetta executed by own virtual machine. Vendetta SDK is high-level abstractions, which has low-level platform-dependent implementations. Implementation provided by VM and will be installed on user's machine at once for all applications.
Very big standart library Vendetta is light-weight language. It means that in most cases You don't need to use third-party libraries. Standard library is very comprehensive: you have pre-built tools for:
  • work with network (TCP, UDP, HTTP 1.1, HTTP/2)
  • file operations
  • graphics & UI
  • work with JSON and XML
  • tools for easy creating RESTful apps
  • work with PUSH-notifications
  • «basic operations»: Math, Text, Date/Time, Logging, Regex, etc.
  • coroutines, threads, proccesses and reactive programming
  • crypto tools: AES, DES, RCA, etc.
  • databases: SQL and DBMS drivers
  • media: hardware acceleration, sound, video, images, 3D graphics
  • and more other features...

Vendetta Language

Fast and simple language for cross-platform development

First low-level cross-platform GUI available in twin-mode Vendetta provides you abstract UI elements, but hides implementation in VM. All virtual machines has two standart GUI implmentations. First based on OpenGL or Vulkan API and allows your app to look the same on all platforms. Second based on OS GUI API and provides you access to platform-native items.
Current status At the moment (may 2020) Vendetta goes through the design and development phase of additional utilities that will facilitate language development in the future. For example, the standard virtual machine implementations for Windows XP+, Linux 3.16, and MacOS Sierra will be based on the Magic library components.
